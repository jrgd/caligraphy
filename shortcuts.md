# Shortcuts Guide

## Keyboard

| Key | Action |
|-----|--------|
| `A-Z`, `0-9` | Select letter to edit |
| `Shift` (hold) | Constrain drawing to axis (horizontal/vertical/45Â°) |
| `Alt` (toggle) | Pen tool mode: click to add points, release to end stroke |
| `Shift + Alt` (hold) | Straight segment drawing |

## Mouse / Trackpad / Stylus

### Draw Mode
| Action | Result |
|--------|--------|
| Click + Drag | Draw continuous stroke |
| Click (with Alt) | Add single point to stroke |
| Multiple clicks (with Alt) | Connect points with line segments |

### Edit Mode
| Action | Result |
|--------|--------|
| Click | Start selection rectangle |
| Click + Drag | Draw selection rectangle |
| Shift + Click | Add to selection |
| Alt + Click | Remove from selection |
| Click on point | Select point |
| Click on stroke | Select entire stroke |
| Drag point | Move point |
| Right-click on point | Delete point |

## Mode Toggle
- Button: Click "Draw Mode" / "Edit Mode" toggle
